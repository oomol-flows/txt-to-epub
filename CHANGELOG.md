# Changelog

## v0.1.3 (最新)
### 智能章节标题增强 ✨
- 自动检测并增强过于简单的章节标题（如仅包含"第X章"）
- 从章节内容中智能提取有意义的标题片段
- 支持中英文双语标题增强
- 提升电子书阅读体验和目录可读性

### 文件编码检测优化 🔧
- 集成 chardet 自动检测文本文件编码
- 支持 UTF-8、GBK、GB2312、GB18030、Big5 等多种编码
- 增强文件读取容错能力
- 优化低置信度编码的回退机制

### 测试流程扩展 🧪
- 添加多个测试节点用于验证不同配置
- 增强 txt-to-epub 子流程的测试覆盖

## v0.1.2
- 国际化支持：为测试流程节点添加 i18n 翻译键

## v0.1.1
- 完善代码文档：优化所有 YAML 文件的描述和元数据
- 清理默认配置：移除冗余的默认值
- 版本发布准备

## v0.1.0
### txt-to-epub 流程 📝
- 智能目录：新增可配置的 TOC 检测阈值和参数
- 错误处理：增强异常处理和日志记录
- 节点扩展：增加第二个处理节点
- 封面优化：移除封面下载脚本，优化图片处理

### core 模块优化 🔧
- 代码重构：完整的 txt-to-epub-core 模块优化升级
